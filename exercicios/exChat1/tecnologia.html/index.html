Tecnologia da Web

Estas são algumas tecnologias usadas no desenvolvimento web.

HTML

HTML é a linguagem usada para estruturar páginas web.

CSS

CSS é usado para estilizar páginas web.

JavaScript

JavaScript permite adicionar interatividade às páginas.

A diferença entre usar "src="logo.png"" e "src="imagens/logo.png"" é:
Um puxa a imagem presente na pasta do onde está o ficheiro index.
E o outro puxa a imagem de outra pasta "imagens".
